⚠️ Effets additifs chimio + RT — Points clés (Table 4 PENTEC)
  • SNC / Neurocognition : Méthotrexate ≈ +5,9 Gy équivalent de dose cérébrale uniforme
  • Moelle épinière : Chimio intrathécale abaisse le seuil de myélopathie (49,7 Gy → 39,6 Gy moyen)
  • Cœur : Chaque +100 mg/m² anthracyclines = HR pour IC = 1,93 ≈ +10,5 Gy dose cœur
  • Rein : Ifosfamide 63 g/m² = 5% risque toxicité sévère sans RT ; avec RT + V10=42% → 10%
  • Audition : Cisplatine 300 mg/m² → déplace la courbe dose-réponse cochléaire de ~7 Gy
  • Sarcomes secondaires : Anthracyclines + alkylants → odds ratio ~2 pour sarcome secondaire
